# Word-to-PDF Images

Place project screenshots in this directory.

Recommended file names:
- `web-ui-screenshot.png` (main README screenshot)
- `conversion-example-before-after.png` (optional)

For root `README.md` / `README.zh-CN.md`, use:

```markdown
![Web UI Screenshot](docs/images/web-ui-screenshot.png)
```

For this folder README (`docs/images/README.md`) itself, use:

```markdown
![Web UI Screenshot](web-ui-screenshot.png)
```
